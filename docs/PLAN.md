# 零硬件具身智能项目包执行计划

## 一句话定位

零硬件具身智能项目包：用开源数据、仿真和 VLA/robot learning 工具，做出可运行、可展示、可写进简历的项目资产。

## 核心判断

这个方向不是空白，但“中文化、零硬件、面向个人付费、交付作品集资产”的产品形态还不成熟。不要和官方课程、开源教程比权威或覆盖面，要比它们更短、更具体、更面向求职结果。

## 免费版目标

免费版不是完整课程，而是信任入口。第一阶段只证明三件事：

1. 用户不用买机器人也能跑出结果。
2. 项目结构清楚，环境可自检。
3. 付费版确实能节省踩坑时间，并补齐简历、报告、面试资产。

## 第一个月节奏

### 第 1 周：免费版 GitHub 仓库

目标：能跑、能展示、能引流。

- 初始化仓库
- 建立项目结构
- 固定环境方案
- 完成环境自检
- 完成数据下载脚本
- 完成基础可视化脚本
- 完成 quick start notebook
- 录制 3 分钟 demo 视频

### 第 2 周：付费版项目包

目标：能卖。

- 补完整 notebook
- 加入 HTML 报告
- 加入数据质量检查
- 加入 FAQ
- 加入简历和面试模板
- 打包 `zero-hardware-lerobot-project-01-v1.0.zip`

### 第 3 周：上架与发布

目标：获得第一批真实购买信号。

- 小报童买断制上架
- B 站发布 1 条长视频

- CSDN/掘金发布 1 篇踩坑文
- 小红书发布 2 篇图文
- GitHub 免费仓库同步更新

### 第 4 周：反馈修复

目标：验证购买意愿，而不是扩张内容。

- 汇总环境报错
- 更新 FAQ
- 统计 GitHub 点击、视频完播、购买转化
- 决定是否推进项目 02
