零硬件 LeRobot 项目包：4 小时跑通你的第一个具身智能项目

不用买机械臂，不用真实机器人。用公开机器人数据集完成 episode 可视化、action 分析、数据质量检查，并生成可写进简历的项目报告。

一个可以展示的具身智能项目：

```text
基于 LeRobot 数据集的机器人 episode 可视化与 action 分布分析系统
```

- 会 Python
- 想入门具身智能 / robot learning / VLA
